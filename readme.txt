Перед использованием данного скрипта нужно установить py желательно версии 3+ ,тк команды ниже я привел только для него
Рекомендую все делать в PyCharm 
В папке программы создаем новый проект , и создаем файл main.py и любое-название.txt 
Открываем терминал и устанавливаем нужные библиотеки, если вы тупо скопируете код, то нажав на знак лампочки вам предложит их установить - качаем
В терминал пропишем py -m pip install pyautogui
Теперь записываем по-строчно наши сообщения в файл любое-название.txt (только английсский транслит) - будет много просьб добавлю возможность отправки русских сообщений и прочих фишек
Запускаем бота ( стоит задержка 5 секунд можно изменить время при изменении числа в строке  3 time.sleep(5)  )
Заходим в нужную соц.сеть нажимаем на полоску отправления сообщений
После 5 секунд бот начинает спамить.
Время спама можно изменить!
